### Usage

#### Install Dependencies (gatsby & sanity)

- cd gatsby
- npm install
- cd sanity
- npm install

#### Run

##### Run frontend (:8000)

- npm start

##### Run frontend with serverless functions

- npm run netlify

##### Run backend (:3333)

- npm start

#### Build

##### frontend

- npm run build

##### backend

- sanity deploy
